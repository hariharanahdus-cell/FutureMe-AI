# 🚀 FutureMe AI

FutureMe AI is an AI-powered web application that predicts a user's future based on their goals, age, and skills. It generates motivational predictions for the next **5, 10, and 15 years** using **Google Gemini AI**.

## ✨ Features
- AI-generated future predictions
- 5, 10 & 15-year roadmap
- Responsive React UI
- Django REST API
- PostgreSQL database
- Fallback prediction when AI is unavailable

## 🛠 Tech Stack
- React + Vite
- Django REST Framework
- PostgreSQL
- Google Gemini AI
- Vercel
- Render
